# 爆裂猫团（BombCatParty）

一个受“抽牌+炸弹+反制连锁”体验启发的原创 Web 卡牌游戏原型：
- 原创名称、原创 UI 风格、原创卡牌文案
- 支持 **人机对战**（1 人 + 1~4 AI）
- 支持 **本地多人 Hotseat**（2~5 人同屏轮流）
- 支持炸弹、拆弹、跳过、连环攻击、预知未来、洗牌、索要、反制连锁、猫咪组合

## 运行方式

推荐使用任意静态服务器：

```bash
cd BombCatParty
python -m http.server 8000
```

然后打开：

`http://localhost:8000`

✅ 当前版本支持直接双击 `index.html` 打开运行（已改为单文件脚本入口，不依赖 ES Module 导入）。

## 文件结构

```text
BombCatParty/
├─ index.html            # 主页面与挂载点
├─ styles.css            # 全局样式、布局、动画、弹窗
├─ README.md
└─ js/
   ├─ standalone.js      # 无模块依赖入口（file:// 直开可用）
   ├─ main.js            # 启动入口、屏幕切换、玩家交互流程
   ├─ engine.js          # 游戏规则与状态机（发牌/回合/抽牌/结算/淘汰/胜利）
   ├─ cards.js           # 卡牌定义与卡池构建
   ├─ ai.js              # AI 决策与反制/拆弹策略
   ├─ ui.js              # UI 渲染、弹窗、规则/设置/教程
   ├─ fx.js              # 动画触发与简易音效管理
   └─ utils.js           # 工具函数
```

## 已实现规则要点

- 初始每位玩家：6 张随机牌 + 1 张拆弹包
- 牌库含：
  - 爆裂猫（炸弹）
  - 拆弹包
  - 跳过、连环攻击、预知未来、洗牌、索要、反制
  - 5 种猫咪收集牌
- 回合推进：玩家可打牌，通常需通过抽牌结束当前小回合
- 连环攻击：使下家额外增加小回合压力
- 反制连锁：可对他人出牌响应，且可连续反制（奇数层取消原牌，偶数层原牌生效）
- 抽到炸弹：自动尝试拆弹；无拆弹则淘汰
- 猫咪组合：两张同名可偷 1 张，三张以上可强化（随机偷更多）
- 最后一名存活者获胜

## 交互与体验

- 主菜单（开始/规则/设置）
- 规则说明弹窗
- 设置项：音效开关、动画速度
- 新手首次进入提示
- 回合日志、剩余牌库计数、当前回合高亮
- 本地多人切换遮罩，防偷看手牌
- AI 行动“思考停顿”提示

## 后续可扩展方向

1. **卡牌扩展包**：陷阱牌、交换手牌、延时炸弹、事件牌
2. **更强 AI**：基于风险评估与记牌概率的策略层
3. **更丰富动画**：卡牌飞行动画、角色表情演出、淘汰镜头
4. **联网模式**：WebSocket 房间与断线重连
5. **存档系统**：战绩、回放、配置云存储
6. **移动端强化**：触控手势、竖屏布局优化
