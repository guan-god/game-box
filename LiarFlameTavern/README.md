# 谎焰酒馆（LiarFlameTavern）

一个原创酒馆诈唬对局 Web 游戏原型，核心体验：
- 盖牌宣称
- 质疑与反心理博弈
- 高压惩罚轮盘
- 最终只剩一人生还

## 文件结构

```text
LiarFlameTavern/
├─ index.html      # 页面入口
├─ styles.css      # 酒馆氛围UI、布局、动画、轮盘演出样式
├─ game.js         # 规则状态机 + AI + UI渲染 + 交互
└─ README.md
```

## 运行方式

可直接双击 `index.html` 打开，或本地静态服务器：

```bash
cd LiarFlameTavern
python -m http.server 8000
```

访问：`http://localhost:8000`

## 核心规则说明（第一版）

1. 每轮有目标牌值（K/Q/A）
2. 当前玩家打出 1~3 张盖牌并宣称“全部是目标值”（可说谎）
3. 下家可选择：
   - 相信：继续流程
   - 质疑：翻开上一组牌检查真假
4. 质疑结算：
   - 若被质疑者说谎：被质疑者进轮盘
   - 若被质疑者没说谎：质疑者进轮盘
5. 轮盘惩罚：6格，命中危险格则淘汰
6. 最后一名存活者获胜

### 特殊牌（第一版）
- 万用牌：可当任意目标值
- 反转牌：改变当前目标值
- 透视牌：提供窥视演出（信息战）
- 烟雾牌：让本次质疑惩罚更危险（危险格增加）
