# 双相夺核：晶格战线

原创“卡牌 + 棋盘走位”对战原型（Hearth-like 节奏 + 实体棋盘单位）。

## 运行

```bash
cd MirrorLockdown
python3 -m http.server 5173
```

浏览器打开 `http://localhost:5173`。

## 当前版本内容

- 棋盘：7x5，对称地形，双方水晶位于两端中央。
- 资源：法力每回合上限 +1（最多10）并回满，后手有幸运币。
- 卡牌类型：随从 / 法术 / 地形。
- 已实现 18 张基础卡（含你指定的 10 张核心卡）。
- 关键词示例：冲锋、突袭、守卫、飞行、护盾、亡语、光环、穿透、连击。
- 模式：PVP 本地双人 / PVE（简单、普通、困难）。
