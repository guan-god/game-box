# 双相夺核（Biphase Core Clash）

紧凑型双人同步结算棋盘博弈 Web 原型。

## 运行

```bash
cd MirrorLockdown
python3 -m http.server 5173
```

浏览器打开 `http://localhost:5173`。

## 新版玩法要点

- 6x6 镜像对称随机地图
- 双方仅 3 单位：核心体 / 突击者 / 干扰者
- 每回合各选 2 张行动牌并同时翻牌结算
- 5 张行动牌：移动 / 突袭 / 冲刺 / 脉冲 / 屏障
- 用过的牌进入 1 回合冷却
- 胜利条件：击破核心 / 3 点中心能量 / 对手卡死一回合
- 支持本地 PVP + PVE（简单/普通/困难）
