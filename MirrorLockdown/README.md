# 双相夺核：晶格战线

原创“卡牌 + 棋盘走位”对战原型（炉石节奏感 + 实体棋盘单位）。

## 运行

```bash
cd MirrorLockdown
python3 -m http.server 5173
```

打开 `http://localhost:5173`。

## 第三轮重构亮点（卡牌深度）

- 卡牌三分法：**随从 / 法术 / 秘术**
- 关键词体系：战吼、亡语、守卫、冲锋、突袭、圣盾、潜行、吸血、远程、飞行、光环、穿透、连击
- 内置两套预设卡组：
  - 晶甲中速（守卫 + 站场 + 战吼/亡语）
  - 秘术控制（法术 + 秘术 + 远程终结）
- 秘术隐藏区与触发反制：攻击水晶、施法、进中心、友方死亡、敌方高费召唤
- 水晶技能：每回合可花 2 法力发动一次（晶能脉冲 / 修复回路）
- 胜利方式：摧毁敌方水晶
