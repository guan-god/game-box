# 镜域封锁（Mirror Lockdown）

原创双人同时行动棋盘策略 Web 游戏原型。

## 运行

直接用任意静态服务器启动：

```bash
cd MirrorLockdown
python3 -m http.server 5173
```

浏览器打开 `http://localhost:5173` 即可。

## 核心特性

- PVP 同屏与 PVE（简单/普通/困难）
- 7x7 / 9x9 镜像对称随机地图
- 同时行动结算 + 回合优先权轮换
- 事件模式（每 3 回合）
- 地形机制：能量格/干扰格/护盾格
- 回合日志与上回合回放
- 动画速度调节、音效开关、移动端基础适配
